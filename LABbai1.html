HTML Tutorial Summary

1. HTML Basic

HTML (Hypertext Markup Language) is the standard language for creating web pages. An HTML document starts with <!DOCTYPE html> to define the document type, followed by the <html> element that contains all the content of the page.

Example:

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

2. HTML Elements

HTML elements are the basic building blocks of HTML pages. An element consists of a start tag, content, and an end tag.

Example:

<p>This is a paragraph.</p>
<a href="https://www.example.com">This is a link.</a>

3. HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

Example:

<a href="https://www.example.com">Visit Example</a>
<img src="image.jpg" alt="Description of image">

4. HTML Headings

HTML headings are used to define headings in an HTML document. There are six levels of headings, from <h1> (most important) to <h6> (least important).

Example:

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

5. HTML Paragraphs

The <p> tag defines a paragraph in an HTML document.

Example:

<p>This is a paragraph of text.</p>

6. HTML Styles

HTML styles are used to add CSS (Cascading Style Sheets) directly within an HTML document using the <style> tag or by adding the style attribute to elements.

Example:

<p style="color:blue;">This is a blue paragraph.</p>
<style>
    body {font-family: Arial, sans-serif;}
</style>

7. HTML Formatting

HTML formatting tags are used to apply formatting to text, such as bold or italics.

Example:

<b>Bold text</b>
<i>Italic text</i>
<u>Underlined text</u>

8. HTML Quotations

The <blockquote> tag is used to define a block of text that is quoted from another source. The <q> tag is used for inline quotations.

Example:

<blockquote>
    This is a block quote.
</blockquote>

<q>This is an inline quote.</q>

9. HTML Comments

Comments in HTML are not displayed in the browser and are used to add notes to the code. They are enclosed in <!-- -->.

Example:

<!-- This is a comment -->

10. HTML Colors

HTML colors can be defined using names, hexadecimal values, RGB values, or HSL values.

Example:

<p style="color:red;">This text is red.</p>
<p style="color:#00FF00;">This text is green.</p>
<p style="color:rgb(0,0,255);">This text is blue.</p>

11. HTML CSS

CSS (Cascading Style Sheets) is used to style and layout HTML pages. CSS can be applied inline, internally within a <style> tag, or externally via a separate CSS file.

Example:

<style>
    p {
        color: red;
        font-size: 20px;
    }
</style>

12. HTML Links

The <a> tag defines a hyperlink, which is used to link from one page to another.

Example:

<a href="https://www.example.com">Visit Example</a>

13. HTML Images

The <img> tag is used to embed images in an HTML page.

Example:

<img src="image.jpg" alt="Description of image">

14. HTML Tables

Tables are defined using the <table> tag. Rows are defined with <tr>, and table headers and data are defined with <th> and <td>, respectively.

Example:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

15. HTML Lists

Lists can be ordered (<ol>) or unordered (<ul>). List items are defined with the <li> tag.

Example:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

16. HTML Block & Inline

Block-level elements take up the full width available, with a new line before and after the element. Inline elements only take up as much width as necessary and do not start on a new line.

Example:

<div>This is a block-level element.</div>
<span>This is an inline element.</span>
